## 使用例

```sh
python stagePowerPointCreator.py ^
    --input-path 【*.png】 ^
    --output-path stage.pptx ^
    --column 5 ^
    --start-x 0.5 ^
    --start-y 0.5 ^
    --offset-x 4.75 ^
    --offset-y 8 ^
    --height 7.5
```
